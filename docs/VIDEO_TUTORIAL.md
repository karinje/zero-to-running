## Creating Visual Content

### Screenshots

Use your preferred screenshot tool:
- macOS: Cmd+Shift+4
- Linux: Screenshot tool
- Windows: Snipping Tool

### GIFs

Tools for creating GIFs:
- [LICEcap](https://www.cockos.com/licecap/) (macOS/Windows)
- [Peek](https://github.com/phw/peek) (Linux)
- [Kap](https://getkap.co/) (macOS)

### Diagrams

Tools for architecture diagrams:
- [Draw.io](https://app.diagrams.net/)
- [Lucidchart](https://www.lucidchart.com/)
- [Mermaid](https://mermaid.js.org/) (text-based)
